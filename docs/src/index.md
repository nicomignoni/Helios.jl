# Helios.jl

*Julia functions for computing Sun-related quantites.*

`Helios.jl` is a collection of pure functions which compute useful quantites for Sun-related calculations, such as [irradiance](https://en.wikipedia.org/wiki/Irradiance) and [solar position](https://en.wikipedia.org/wiki/Position_of_the_Sun). 

## Installation
Install `Helios.jl` from the Julia REPL

```julia
add ] https://github.com/nicomignoni/Helios.jl.git
```
