Irradiance

Helios.Irradiance — Type
Irradiance(dni, dhi, ghi)

Container for solar irradiance components.

Fields

  • dni: Direct Normal Irradiance [W/m²].
  • dhi: Diffuse Horizontal Irradiance [W/m²].
  • ghi: Global Horizontal Irradiance [W/m²].
